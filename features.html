VSCode's Features

Multi-line editing

VSCode has support for multi-line editing. Hold down Alt and click somewhere in the file, it will add another cursor to that position, then when you type it will type it in both places.

Emmet

Emmet is a tool to make writing HTML (XML in general) easier and more efficient. It has helpful macros and shortcuts to speed up HTML development.

If you type h1#title+p.red+ul>li*5 and hit Enter, it will expand to this:

<h1 id="title"></h1>
<p class="red"></p>
<ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>
Here's a breakdown of what it did

h1

This creates an h1 element

#title

This gives an id of "title" to the h1 element

+p

This makes a p element next to the h1 element

.red

This gives a class of "red" to the p element

+ul

This makes a ul element next to the p element

>li

This makes an li element inside the ul element

*5

This repeats the li element 5 times

Git
